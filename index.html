Apa itu Website

Website merupakan halaman informasi yanng disediakan melalui jalur internet, sehingga bisa diakses diseluruh dunia selama terkoneksi dengan jaringan internet. Sebuah situs web biasanya ditempatkan pada sebuah server web yang dapat diakses menggunakan jaringan (Internet atau LAN) melalui URL. Sebuah halaman web adalah berkas plain text yang diatur dan dikombinasikan sedemikian rupa dengan instruksi-instruksi berbasis HTML atau XHTML, terkadang disisipi dengan bahasa skrip. Berkas tersebut kemudian diterjemahkan oleh peramban web dan ditampilkan seperti layaknya sebuah halaman pada monitor komputer. Halaman-halaman web tersebut diakses oleh pengguna melalui protokol komunikasi jaringan yang disebut sebagai HTTP.

Canicomp

Canicomp adalah salah satu ekstrakurikuler di SMA Kanisius yang membahas algoritma komputer dan membuat proyek-proyek komputer skala kecil. Ekstrakurikuler ini bertujuan untuk memperkenalkan siswa pada konsep-konsep dasar pemrograman dan algoritma, serta memberikan kesempatan kepada siswa untuk mengembangkan keterampilan teknis mereka dalam membuat proyek-proyek komputer skala kecil. Dalam Canicomp, siswa akan belajar tentang berbagai bahasa pemrograman seperti Python dan Javascript, serta mempelajari konsep-konsep seperti struktur data dan algoritma pengurutan. Selain itu, siswa juga akan belajar bagaimana membuat program sederhana dan membangun proyek-proyek seperti game sederhana dan website.
